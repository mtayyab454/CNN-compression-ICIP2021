### Model Compression

Following commands can be used to reproduce the results presented in the paper. 

##### 1. Resnet56

| Flops         | Parameters      | Accuracy |
|---------------|-----------------|----------|
|89.80M(64.22%) | 0.32M(62.97%)   | 92.71%   | 

```shell
python python train_cifar.py \
--jobid resnet56_test \
--arch resnet56 \
--dataset cifar10 \
--compress-rate 0.63 \
--l2-weight 0.001 \
--add-bn True \
--epochs 120 \
--schedule 30 60 90 \
--lr 0.01
```
